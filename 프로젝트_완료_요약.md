## 🚀 즉시 실행 가능한 3단계

### Step 1: 로컬 테스트 (5분)
```bash
cd /mnt/user-data/outputs/pyeongsu-calculator-site
npm install
npm run dev
```
→ http://localhost:3000 접속

### Step 2: 도메인 구매 (오늘)
- **권장**: pyeongsu-calculator.kr ($12/년)
- **대안**: pyeongcalc.com ($10/년)
- **구매처**: Gabia 또는 Namecheap

### Step 3: 배포 (내일)
```bash
npm run build
```
→ Cloudflare Pages에 out/ 업로드
